## Python - příprava dat
V této složce se nachází skripty použité pro stažení a přípravu dat použitých v projektu.

V toto chvíli nejsou ve finálním stavu a před jejich spuštěním doporučuji si kód prostudovat.
Např. `download.py` se pokusí stáhnout všechny části digitálního modelu terénu pro celou ČR (~40 GB), což nemusí být vždy žádoucí.

Pro spuštění samotného projektu jsou všechna data již připravena a není nutné tyto skripty spouštět.
