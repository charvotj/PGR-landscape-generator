# Projekt do PGR - Generování reálné krajiny se zaměřením na lesy
V tomto repozitáři je kód a minimální potřebná data pro jeho spuštění. Pro zobrazení jiné části mapy nebo jiných modelů stromů je potřeba dotyčná data zvlášť připravit. 
Plně automatické stažení všech dat nakonec nebylo součástí tohoto projektu.

# Jak spustit?
Na linuxu následujícím způsobem:
```
mkdir build
cd build
cmake ..
make
.\PGR
```

## Zdroje
- Generování modelů stromů - [https://github.com/friggog/tree-gen](https://github.com/friggog/tree-gen)
- Materiály a textury - [https://www.blenderkit.com/](https://www.blenderkit.com/)
